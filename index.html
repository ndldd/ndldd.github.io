www.TrackerGame.de

Tracker Game is a single-page web application that allows a user to track his daily food energy uptake. It offers various ways to create, manage and categorize food entries. It features games elements, like a quest and task system, achievements, tutorial, short and longterm progressindicators. And assists him optimzied for devices with small screens.
  • Business idea: Gamification of an calorie counnter. Use of gamification mechanisms of online social and casual games.To great a more effective weightloss tools Lower the treshold make weight loss easier, giving keeping the use rewarding and motivating
  • Design and implementation: Single-page web application. Dependency injection mechanism of angular separate business logic and modular way. Model classes live in respective services, and used inheritance to avoid duplicaton. decoupling a publish subscribe service Responsiveness I use bootstrap framework (Grid-layout system) app itself is fully responsive and optimized for mobile platforms single page web application larger project optimized for mobile, development server and link to the live site.
  • Libraries and frameworks used: Frontend: AngularJS, Bootstrap3, D3; Backend: Python Django and Django REST framework, Nginx, PostgreSQL, (a version using Neo4j as a graph database is currently in development)
  • Link to live site: www.trackergame.de (registration / marketing site)
