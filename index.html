www.TrackerGame.de

About the Project: Tracker Game is a web application that allows a user to track his daily food energy uptake. It offers several ways to create, manage and categorize food entries. It combines these tracking features with... In addition it features various games elements like a quest and task system, reward popups, achievements, tutorial, short and long-term progress indicators. The layout is responsive and optimized for devices with small screens.
  • Motivation for this project was to create a more effective weight loss tool. mechanism to influence behavior used in social / casual games today. With the measurement of an caloire counter. self control combining effective mechanism to influence behavior from games with an effective method to control weight. more rewarding easier to sustain motivation. attracitve game them lower the entrance borader. Gamification of an calorie counnter. Use of gamification mechanisms of online social and casual games.To great a more effective weightloss tools
  • Design and implementation: Html5 JavasScript Single-page web application. Dependency injection mechanism of angular separate business logic and modular way. Model classes live in respective services, and used inheritance to avoid duplicaton. decoupling a publish subscribe service Responsiveness I use bootstrap framework (Grid-layout system) app itself is fully responsive and optimized for mobile platforms single page web application larger project optimized for mobile, development server and link to the live site.
  • Libraries and frameworks used: Frontend: AngularJS, Bootstrap3, D3; Backend: Python Django and Django REST framework, Nginx, PostgreSQL, (a version using Neo4j as a graph database is currently in development)
